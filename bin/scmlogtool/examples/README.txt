TAC SCM Log Toolkit Example
---------------------------

This is a quickly compiled README file.  Please bear with it.

This package contains a simple example of a simple log analyzer for
TAC SCM log files. It simply counts all offers sent by suppliers and
manufacturers.


Requirements
------------
The TAC SCM Log Toolkit and Java SDK 1.4.


Compiling
---------
Type "compile.bat" (or "compile.sh" under Unix) to compile the example.


Running the example
-------------------

1. Download a game log file from any of the TAC SCM servers.

2. Type "runexample.bat -file game.slg.gz"
   (or "runexample.sh -file game.slg.gz" under Unix)
